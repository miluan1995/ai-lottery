# AI Lottery Token — Design Spec

## 概念
BSC 链上首个 AI Agent 彩票代币。基于 Flap.sh 蝴蝶预言机（Butterfly Oracle），每小时 AI 自动开奖。

## 代币机制
- 平台: Flap.sh (BSC)
- 税率: 10% Tax Token
- 税收分配:
  - 5% → 当期奖池
  - 4% → 下期奖池预存
  - 1% → Dev 钱包 (0xD82913909e136779E854302E783ecdb06bfc7Ee2)

## 开奖规则
- 频率: 每 1 小时开奖一次
- 奖金: 奖池余额的 60%，直接全发给赢家（无锁仓）
- 剩余 40% 留底 + 下期预存持续注入 = 奖池永不归零

## AI 开奖逻辑（蝴蝶 Oracle）
AI 从合格持币者中选出赢家，评分依据：
1. 持币量（开根号加权，防鲸鱼通吃）
2. 持币时间（连续持币加成）
3. 链上活跃度
4. 随机因子

## 连续持币加成
- 持币 ≥ 3 期（3小时）: 中奖概率 ×1.5
- 持币 ≥ 12 期（12小时）: 中奖概率 ×2.0
- 卖出后重新买入: 加成归零重新计算

## 防砸盘机制
- 开奖后 10 分钟内卖出: 税率临时提高到 20%（额外 10% 进奖池）
- 奖池永远只发 60%，保证下期有底
- 9% 税收持续注入奖池，越滚越大

## 合约架构
1. Tax Token — 在 flap.sh 发行，10% 税
2. LotteryVault — 自定义 Vault（继承 VaultBaseV2），处理税收分配 + 开奖逻辑
3. FlapAIProvider — 蝴蝶 Oracle 接入，AI 决策开奖

## 官网功能
- 实时代币数据（价格、市值、持币人数、销毁量）
- 奖池实时余额 + 下期预估奖金
- 开奖倒计时
- 开奖历史（赢家地址、金额、AI 推理摘要）
- 规则说明页
- 持币加成查询（输入地址查倍率）
- 赛博朋克 / 霓虹风格 UI

## 技术栈（官网）
- 纯前端: HTML + CSS + JS（单页应用）
- 链上数据: ethers.js 直连 BSC RPC
- 无后端依赖
